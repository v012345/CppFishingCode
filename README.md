# 纯C++ 捕鱼游戏  
语言:C++  
编辑器:VS2015  
渲染:D2D  
碰撞:obb和aabb  
编译选项:请选择Release x86 编译
## 项目文件目录说明
img 放游戏所使用的图片文件夹  
frame 放控制游戏场景相关的文件  
instance 放游戏窗口进程 游戏循环 工具类 碰撞类的文件  
render 放游戏渲染引擎的文件夹  
trigger 放对一些消息进行映射的文件夹

## 编译说明  
1. 源码clone下来我们新建一个空项目  
2. 在空项目里面加入我们的源码 并包括到项目中  
3. 然后就是进行设置了 **属性 > C/C++ > 语言 > 符合模式**设置为否
4. 然后就是进行设置了 **属性 > 常规 > 字符集** 设置为 `使用 Unicode 字符集`  
5. 然后进行编译
